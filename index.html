GitHub song setup

  1. Create a folder named songs in your GitHub repo.
  2. Upload your audio files there, like songs/song1.mp3.
  3. Add a file named songs/manifest.json that lists those songs.
  4. The app will auto-load them when it starts.
{
  "songs": [
    { "title": "Thunder", "artist": "Team Mix", "file": "song1.mp3" },
    { "title": "Fireball", "artist": "Pitbull", "file": "fireball.mp3" }
  ]
}

Only use audio files you have the right to use.
