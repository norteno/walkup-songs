GitHub song setup

  1. Create a folder named songs in your GitHub repo.
  2. Upload your audio files there, like songs/cook.mp3.
  3. Add a file named songs/manifest.json that lists those songs.
  4. The app auto-loads them when it starts.
{
  "songs": [
    { "title": "COOK", "artist": "Sofi Tukker", "file": "songs/Cook.mp3" },
    { "title": "All I Do Is Win", "artist": "DJ Khaled", "file": "songs/All I Do Is Win.mp3" }
  ]
}

Use the exact filename and capitalization from GitHub.
