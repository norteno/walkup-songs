Setup

  1. In your Spotify Developer Dashboard, add http://127.0.0.1:3000/ as a Redirect URI.
  2. Serve this folder locally with python3 -m http.server 3000.
  3. Open http://127.0.0.1:3000/ in your browser and connect Spotify.

Client ID is already filled in for you. If you ever change it, edit app.js.
